# 67-asm

67-asm is an experimental custom assembly language, assembler, and virtual machine written in C++.

The project is designed to explore:
- assembler design
- virtual machine architecture
- instruction encoding
- executable formats
- memory models
- low-level systems programming

Rather than targeting a real CPU like x86, 67-asm defines its own instruction set architecture (ISA) and runtime.

---

# Goals

- Build a custom assembler
- Build a custom VM/runtime
- Learn systems programming deeply
- Experiment with CPU and ISA design
- Potential future JIT/compiler support
